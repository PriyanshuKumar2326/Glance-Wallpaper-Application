# Glance-Wallpaper-Application
Developed a React-based web application that fetches and displays random high-quality images using a public API. Implemented asynchronous API calls, responsive UI, loading states, and image refresh functionality.

---

## 🚀 Features

- Display random images instantly
- Fetch images from an external API
- Responsive and modern user interface
- Fast loading experience
- Simple and user-friendly design

---

## 🛠️ Tech Stack

### Frontend

## 📥 Installation

### 1. Clone the Repository

Clone or download this repository to your local machine.

### 2. Install Dependencies

Install all required dependencies in both the **Client** and **Server** folders.

### 3. Configure Environment Variables

Create a `.env` file inside the **Server** folder and add your required environment variables such as:

- Server Port
- API Key

---

## ▶️ Running the Application

1. Start the backend server.
2. Start the frontend development server.
3. Open the application in your browser using the local development URL.

Make sure both the frontend and backend are running at the same time.

---

## ✨ How It Works

1. The frontend sends a request to the backend.
2. The backend fetches a random image from the external API.
3. The API response is returned to the frontend.
4. The frontend displays the image along with its related information.
